# FashionHub

A modern and clean e-commerce product page built with HTML, CSS & JavaScript.

Built with semantic HTML5, meticulously styled CSS3, and enhanced with JavaScript for interactivity, this project focuses on delivering a visually immersive and highly responsive user experience. It demonstrates a commitment to modern design principles, clean code, dynamic UI behavior, and aesthetic excellence

## 📸 Preview

You can view a live demo of this project here: [FashionHub](https://mamalireza.github.io/sneakers/)

![FashionHub](https://github.com/Mamalireza/sneakers/blob/main/preview.png)

## 🛠️ Tech Stack

**HTML5:** For semantic content structure.

**CSS3:** For all styling, layout, responsiveness, and design elements.

**JavaScript:** Handles interactive elements such as dynamic content updates, user interactions, and UI behavior.
